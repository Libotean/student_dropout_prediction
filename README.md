# Student Dropout Prediction
Se vor folosi tehnici de machine learning pentru a prezice daca un student va abandona sau nu facultatea, pe baza unor factori precum notele, prezenta la cursuri sau situatia financiara.

## Dataset
**Sursa:** [Student Dropout Prediction Dataset](https://www.kaggle.com/datasets/meharshanali/student-dropout-prediction-dataset)
descarcat la data de: 12.03.2026

**Variabila tinta:** Dropout (0 = student in facultate, 1 = student care a abandonat) - rata de abandon in dataset este de 23.5%.

Principalele caracteristici:
- Date demografice: varsta, gen, venit familial, educatia parintilor
- Comportament academic: ore de studiu pe zi, rata de prezenta, intarzieri la teme
- Performanta: GPA, Semester GPA, CGPA
- Factori externi: job part-time, bursa, acces la internet, timp de naveta, nivel de stres
